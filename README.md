# MY AI Workbench

A personal lab for experimenting with mini LLMs, transformers, agents, and applied AI while learning and building AI systems from scratch.

## 🚀 About

This repository contains my experiments, notes, and implementations related to Artificial Intelligence and Machine Learning, with a strong focus on:

- Large Language Models (LLMs)
- Transformer architecture
- Mini LLM implementations
- AI agents
- Applied AI projects
- AI engineering concepts

The goal of this repo is to **learn by building** and understand how modern AI systems work internally.

## 🧠 Topics Covered

- Tokenization
- Embeddings
- Attention mechanism
- Self-attention (Q, K, V)
- Transformers
- Positional embeddings
- Normalization (RMSNorm / LayerNorm)
- Feed Forward Networks
- LLM inference
- AI agents
- Applied AI tools
